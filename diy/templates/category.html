{% if category == '护肤品' %}
关于护肤品

护肤品是指用于面部和身体护理的化妆品，包括面霜、精华液、乳液、爽肤水等。这些产品能够清洁、保护、营养和美化肌肤，帮助维持肌肤的健康状态。

{% elif category == '面膜' %}
关于面膜

面膜是一种敷在脸上的美容护肤品，具有深层清洁、保湿、美白、抗衰老等功效。根据功效和质地不同，可分为清洁面膜、保湿面膜、美白面膜、抗衰老面膜等。

{% elif category == '护发产品' %}
关于护发产品

护发产品包括洗发水、护发素、发膜、护发精油等，用于清洁、滋养、保护和美化头发。这些产品能够改善头发的质地，增加光泽，预防头发损伤。

{% elif category == '精油' %}
关于精油

精油是从植物中提取的挥发性芳香化合物，具有多种功效。在化妆品中，精油常用于提供香味、抗菌、舒缓、保湿等功能，是天然化妆品的重要成分。
